# H03 — Storage de anexos seguro

Descrição:
Implementar serviço de armazenamento para os documentos enviados (frente/verso), garantindo controle de acesso, paths persistidos no DB e geração de URLs assinadas para download.

Critérios de Aceite:

- Armazenamento configurável (S3 ou filesystem protegido).
- Links assinados com expiração para download.
- Respeitar limites de tipo e tamanho de arquivo.
