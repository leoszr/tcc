# H01.1 — Inicializar repositório Spring Boot (esqueleto)

Objetivo:
Criar o esqueleto do backend com Spring Boot usando o sistema de build definido (Maven ou Gradle). Incluir apenas dependências mínimas necessárias para desenvolvimento do MVP: `spring-boot-starter-web`, `spring-boot-starter-data-jpa`, driver PostgreSQL, `spring-boot-starter-test`.

Entradas / Saídas:

- Entrada: nenhuma
- Saída: repositório backend com `pom.xml`/`build.gradle`, estrutura de pacotes base, aplicação bootstrap (classe main) e configuração básica de `application.properties`/`application.yml` para perfis dev.

Critérios de Aceite:

- Projeto Spring Boot inicia localmente no perfil dev sem erros.
- Arquivo de build contém as dependências listadas.
- README/back-end contém comando para iniciar a aplicação e instruções mínimas (porta, perfil).

Dependências:

- Convenção de nomes de pacotes definida pelo time.

Estimativa: 0.5d (4 horas)

Checklist de QA:

- Clonar o repositório, executar build e iniciar a aplicação; verificar que o processo sobe e um health endpoint responde.
