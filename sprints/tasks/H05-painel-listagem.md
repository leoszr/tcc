# H05 — Painel: listagem e visualização de reservas

Descrição:
Criar as telas do painel da escola para listagem de reservas (filtrável por status) e a tela de detalhe que apresenta todos os dados da solicitação e links temporários para os anexos.

Critérios de Aceite:

- Lista com paginação e filtro por status (PENDING, APPROVED, REJECTED).
- Detalhe exibe campos e links para download de anexos (links assinados).
- Ações rápidas para aprovar/rejeitar a partir do detalhe.

Estimativa: 4d
Responsável: Dev 5
